## Personal Portfolio Website
A professional personal portfolio website designed to showcase my journey as a Computer Information Systems student and aspiring Computer Engineer. This project focuses on clean UI/UX, responsive design, and modern web functionality.

## Features
- Dark Mode Toggle: A user-controlled theme switcher using CSS variables and JavaScript.
- Scroll-Reveal Animations: Dynamic content visibility that triggers as the user scrolls down the page.
- Smooth Navigation: Internal links with smooth-scrolling behavior for a seamless user experience.
- Responsive Layout: A mobile-friendly design utilizing a sticky navigation bar and flexible containers.

## Technologies
- HTML5: Semantic structure for better accessibility and SEO.

- CSS3: Custom properties (variables), transitions, and flexbox for styling.

- JavaScript (ES6): Logic for theme switching, scroll detection, and navigation.

## How to Run
1. Clone the repository:
git clone https://github.com/thaivo6972/portfolio-website.git

2. Open the project:
Simply open the index.html file in any modern web browser.

## What I Learned
- DOM Manipulation: Using JavaScript to toggle classes and handle scroll events.

- CSS Architecture: Implementing a theme system using root variables for light/dark modes.

- UI/UX Design: Engineering a consistent navigation schema and layout to optimize user flow.

- Accessibility: Using semantic tags and ARIA labels to ensure baseline compliance.
